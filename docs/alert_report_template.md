# 告警报告模板使用说明

## 概述

告警报告模板（`alert_report`）是专门为系统告警报告、监控报告设计的专业模板。该模板具有以下特点：

- **专业的配色方案**：使用红色和橙色突出告警信息
- **清晰的层级结构**：适合展示多级标题和分类信息
- **优化的表格样式**：红色表头，适合展示告警统计数据
- **正式的文档格式**：包含页眉页脚、页码等专业元素

## 使用方法

### 在 Dify 工作流中使用

在调用 `markdown_to_word` 工具时，将 `templates` 参数设置为 `alert_report`：

```json
{
  "markdown_text": "# 2025年10月主机告警信息分析报告\n\n...",
  "templates": "alert_report",
  "font_family": "微软雅黑",
  "font_size": 11,
  "line_spacing": 1.5,
  "page_margins": 2.5,
  "paper_size": "A4",
  "output_file": "alert_report.docx",
  "add_page_numbers": true
}
```

### 模板特性

#### 1. 标题样式
- **H1**：居中，18号字体，底部红色边框（2pt）
- **H2**：左对齐，15号字体，红色文字，底部橙色边框（1pt）
- **H3**：左对齐，13号字体，黑色文字

#### 2. 表格样式
- **表头**：红色背景（#f44336），白色文字，加粗
- **表格边框**：1pt，深灰色（#424242）
- **单元格内边距**：6pt
- **交替行颜色**：浅橙色背景（#fff3e0）

#### 3. 配色方案
- **主色调**：红色（#d32f2f）- 用于严重告警
- **次要色**：橙色（#ff9800）- 用于警告告警
- **强调色**：蓝色（#2196f3）- 用于信息告警
- **正常色**：绿色（#4caf50）- 用于正常状态

#### 4. 页眉页脚
- **页眉**：显示"告警报告 - {标题}"，底部红色边框
- **页脚**：显示"机密文档 - 仅供内部使用"，顶部灰色边框
- **页码**：格式为"第 X 页 / 共 Y 页"

## 示例 Markdown 结构

```markdown
# 2025年10月主机告警信息分析报告

---

## 一、短信告警情况汇总

2025年10月主机系统运行整体安全可控...

| 告警分类       | 告警子类         | 10月告警量（条） |
|----------------|------------------|------------------|
| 系统监控告警   | CPU使用率过高    | 29               |
|                | 内存使用率过高   | 46               |

---

## 二、异常告警分析

### （一）硬件监控告警

10月硬件监控类告警共2条...

| 序号 | 应用系统       | 告警描述                    | 级别 | 处理进度 |
|------|----------------|-----------------------------|------|----------|
| 1    | 数据备份平台   | 存储阵列LUN容量达90%阈值     | 4    | 扩容完成 |
```

## 适用场景

- 系统监控告警报告
- 运维分析报告
- 故障分析报告
- 性能监控报告
- 安全告警报告

## 注意事项

1. 建议使用 **微软雅黑** 字体，以获得最佳显示效果
2. 表格数据较多时，建议使用该模板以突出重要信息
3. 告警级别建议使用表格展示，便于统计和分析
4. 文档较长时，建议启用目录（TOC）功能
